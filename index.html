Ваш список докидки порожній.

Перейдіть на вкладку «Товари» та відмітьте галочкою потрібну продукцію.

Введіть сюди URL веб-додатку Google Apps Script для автоматичного запису у вкладку «Докидка відкладеного».

Проста інструкція з підключення (робиться за 1 хвилину):

  1. Відкрийте вашу Google Таблицю.
  2. Вгорі оберіть Розширення (Extensions) ➔ Apps Script.
  3. Видаліть увесь стандартний код і вставте скрипт за кнопкою нижче:
  1. Натисніть кнопку Деплой (Deploy) ➔ Нове розгортання (New deployment).
  2. Оберіть тип розгортання: Веб-додаток (Web app).
  3. У полі «Хто має доступ» обов'язково вкажіть: Усі (Anyone) (це безпечно, дані пишуться лише у ваш файл).
  4. Натисніть Розгорнути (Deploy) та скопіюйте отриманий URL веб-додатка в поле вище.
